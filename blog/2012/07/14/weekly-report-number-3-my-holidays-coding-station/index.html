Weekly Report #3 : My Holidays Coding Station

Jul 14th, 2012

As the title suggests, I’m on holidays in the French Alps. This doesn’t mean that my GSoC is on hold, but it means that I’m enjoying a beautifull sight while coding :

During this week, I’ve continued my work on EmpathyRosterModel and EmpathyRosterModelManager. The model now catches all the signals emitted by the manager previously catched by the view. It also now have all the functions needed to be able to completely rely on the model and remove every manager references in the view. As said in my previous post, all of this is pushed on github : lcontzen/empathy.

On a side note, I found out that if all your top contacts are offline the group name isn’t displayed and the top contacts are lost alone on the top of the roster. I reported it and submitted a patch fixing this behavior (bug #679868) (this still needs to be reviewed).

PS : Good news, everyone! I passed my Midterm Evaluation! \o/
